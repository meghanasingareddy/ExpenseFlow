# **App Name**: ExpenseWise

## Core Features:

- Dashboard Summaries: Display summary cards for total balance, income, and expenses.
- Budget Tracking: Visualize monthly spending budget with a horizontal progress bar.
- Spending Breakdown: Present spending categories using a pie chart with a color-coded legend.
- Transaction History: List recent transactions with key details like icon, title, date, and value.
- Theme Toggle: Allow users to switch between light and dark mode for personalized viewing.
- AI Insights: Use a tool to provide AI-driven insights on spending habits and potential savings.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a sense of stability and trustworthiness.
- Background color: Light Gray (#F5F5F5) for a clean, modern look. In dark mode, very dark gray (#1A1A1A).
- Accent color: Teal (#009688) for interactive elements and highlights, providing a vibrant contrast.
- Body and headline font: 'Inter', a sans-serif font for a modern, neutral look.
- Use a consistent set of line icons to represent transaction types and categories.
- Employ rounded corners and soft shadows for a gentle, approachable interface.
- Subtle transitions and animations for a smooth user experience when navigating and loading data.
